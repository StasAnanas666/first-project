1. Создать папку для проектов.
2. Перейти в нее в VSCode(File -> Open Folder).
3. Открыть терминал(... -> Terminal -> New Terminal или Ctrl+Shift+` или потянуть за панель инструментов внизу, синяя строчка).
4. В терминале выполнить команду npx create-react-app project-name, где project-name название проекта без больших букв.
5. После установки выбрать File -> Open Folder, выбрать папку созданного проекта.
6. В терминале выполнить команду npm start, ждать запуска проекта в браузере.
